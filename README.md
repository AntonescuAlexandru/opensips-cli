# OpenSIPS CLI (Command Line Interface)

OpenSIPS CLI is a tool used to control **OpenSIPS SIP server**.

## History

This project was started by **Dorin Geman** ([dorin98](https://github.com/dorin98)) as part of the [ROSEdu 2018](http://soc.rosedu.org/2018/) program. It has later been adapted to the new OpenSIPS 3.0 MI interface and became the main external tool for managing OpenSIPS.
